Hospital Scheduling Application

Administrative Dashboard

Welcome to the Hospital Scheduling Application. This system is designed for administrative staff to manage patient appointments, staff assignments, and medical records.

System Overview

Patient Management

  • View All Patients: Browse and search through all patient records
  • Filter Patients: Search by name, age group (adult/child), birth year range
  • Create New Patient: Add new patients to the system with demographic and contact information
  • Update Patient Records: Modify existing patient information including address, primary care doctor, and profile picture
  • View Patient History: See all appointments for any individual patient

Doctor Management

  • View All Doctors: Browse all doctor records in the system
  • Create New Doctor: Add new doctors with specialty information and profile pictures
  • Update Doctor Records: Modify doctor information including specialty and profile picture
  • Assign to Appointments: Assign single or multiple doctors to appointments

Nurse Management

  • View All Nurses: Browse all nurse records in the system
  • Create New Nurse: Add new nurses with profile pictures
  • Update Nurse Records: Modify nurse information and profile pictures
  • Assign to Appointments: Assign single or multiple nurses to appointments

Appointment Management

  • View All Appointments: Browse all scheduled appointments
  • Filter Appointments: Search by appointment type, assigned doctor, assigned nurse, patient name, or date range
  • Create New Appointment: Schedule appointments from:
    • Day schedule view (click time slot)
    • Patient detail page (pre-fills patient)
    • Direct creation form
  • Delete Appointments: Remove appointments with confirmation (accessible from appointment detail page only)
  • Appointment Types Supported: Checkups, Post-Operation, Specialist Consultation, Follow-up Visits, Nurse Appointments, Pre-operation Visits, Counseling Sessions, Telehealth Visits, Vaccinations, Annual Checkups, Birth Control Consultations, Medication Checks

Calendar Views

  • Month Calendar View: Visual calendar showing all appointments for a selected month
    • Navigate between months
    • Click any day number to see detailed day schedule
    • See appointment summaries on each day
  • Day Schedule View: Hour-by-hour breakdown of appointments for a specific day
    • Time slots from 8 AM to 8 PM
    • Add appointments directly from any time slot
    • See patient, doctor, and nurse information for each appointment
    • Navigate between days
